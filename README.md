# self-study-and-practice

Self-study & practice repo covering software engineering fundamentals — data structures, OOP, algorithms (LeetCode practice), and data science. Most contents are organized under the `โปรเเกรม` (Program) directory.

## Project Structure

- `fundamental/`: Python fundamentals practice — building core skills for problem solving and interview prep.
  - `learn basic/`: Reference material (Python textbook PDF).
  - `pratice coding/python/`: Hands-on exercises covering core syntax and concepts: variables, loops (`for`/`while`), functions, modules, classes/OOP (`bankAccount.py`), exceptions, tuples/lists, string handling, `match` pattern matching (capture, guard, OR, sequence, mapping patterns), and login/auth logic exercises.

- `โปรเเกรม/`
  - `fundamental/`: Basic programming concepts and early practice.
  - `algorithms/`: LeetCode-style algorithm practice (sorting, searching, data structure problems) — daily/session-based practice files.
  - `data_structures/`: Common data structures implementation.
  - `python_basics/`: Python specific syntax and basic exercises.
  - `data_science/`: Data analysis and science practice (Pandas, Numpy).
  - `machine_learning/`: Machine learning experiments (Scikit-Learn).
  - `fastapi_practice/`: Web development with FastAPI.
  - `nestjs_basics/`: Backend development with NestJS.
  - `postgres_db/`: SQL and PostgreSQL practice.
  - `docker_practice/`: Containerization practice.
  - `basicdart/`: Dart programming practice.
  - `enrich.py`: Utility script to populate practice files with snippets.
  - `time_machine.ps1`: PowerShell script for simulating historical practice commits.

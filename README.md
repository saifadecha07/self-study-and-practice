# self-study-and-practice

Project structure for self-study and practice coding. All contents are organized under the `โปรเเกรม` (Program) directory.

## Project Structure

- `โปรเเกรม/`
  - `fundamental/`: Basic programming concepts and early practice.
  - `algorithms/`: Implementation of various algorithms.
  - `data_structures/`: Common data structures implementation.
  - `python_basics/`: Python specific syntax and basic exercises.
  - `data_science/`: Data analysis and science practice (Pandas, Numpy).
  - `machine_learning/`: Machine learning experiments (Scikit-Learn).
  - `fastapi_practice/`: Web development with FastAPI.
  - `nestjs_basics/`: Backend development with NestJS.
  - `postgres_db/`: SQL and PostgreSQL practice.
  - `docker_practice/`: Containerization practice.
  - `basicdart/`: Dart programming practice.
  - `enrich.py`: Utility script to populate practice files with snippets.
  - `time_machine.ps1`: PowerShell script for simulating historical practice commits.
